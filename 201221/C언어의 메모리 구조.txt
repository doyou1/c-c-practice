    코드 영역       ^
    데이터 영역     |   운영체제의 의해
    힙 영역         |   할당된 메모리 공간
    스택 영역       v
    
    - 코드 영역
    : 실행할 프로그램의 코드가 저장되는 메모리 공간.
     CPU는 코드 영역에 저장된 명령문들을 하나씩 가져가서 실행함

    - 데이터 영역
    : 전역변수와 static으로 선언되는 static 변수가 할당됨
     이 영역에 할당되는 변수들은 프로그램의 시작과 동시에
     메모리 공간에 할당되어 프로그램 종료시까지 남아있게 된다는 특징

    - 스택 영역
    : 지역변수와 매개변수가 할당됨. 이 영역에 할당되는 변수들은
    선언된 함수를 빠져나가면 소멸된다는 특징

    - 힙 영역
    : 데이터 영역에 할당되는 변수와 스택 영역에 할당되는 변수들은 생성과 소멸의 시점이
    이미 결졍되어 있음. 그러나, 다른 성격의 변수가 필요하기도 함.
    그래서 C언어는 원하는 시점에 변수를 할당하고, 소멸하도록 지원함.
    이러한 유형의 변수들이 할당되는 영역이 힙 영역
